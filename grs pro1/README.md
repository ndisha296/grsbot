# GRS Pro1

A Python project.

## Setup

1. Ensure Python is installed.
2. Create and activate virtual environment:
   - The project has a configured venv at `.venv/`
   - To activate: `.venv\Scripts\activate` (Windows)
3. Install dependencies:
   - `pip install -r requirements.txt`

## Usage

Run the main script:
- `python main.py`
